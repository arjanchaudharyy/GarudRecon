## Summary

### Key Components
1. **Flask Backend** - Python web server with platform detection
2. **Platform Detection** - Identifies OS (Windows/Linux/Mac)
3. **WSL Integration** - Automatic bash script execution on Windows
4. **Scan Scripts** - Bash scripts for reconnaissance
5. **Web UI** - Browser-based interface

### Windows-Specific Features
- UTF-8 encoding configuration
- WSL availability check
- Automatic command translation
- Path conversion (Windows â†’ WSL)
- Clear error messages

### Execution Modes
- **Linux/Mac**: Native bash execution
- **Windows + WSL**: WSL-wrapped execution
- **Windows (no WSL)**: Error with installation guide
- **Docker**: Containerized (all platforms)

This architecture ensures GarudRecon works seamlessly across all platforms while maintaining the bash-based scanning engine.
